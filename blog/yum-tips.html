YUM Tips

December 2012 · 1 minute read

yum 很多操作需要 yum-utils 包，我们使用下面的命令安装好这一依赖：

yum -y install yum-utils

  • 列出给定包中所包含的文件

    repoquery -lq your_package_name_here

    How to list the contents of a package using YUM?

  • 下载 srpm (source rpm package)

    yumdownloader --source your_package_name_here
